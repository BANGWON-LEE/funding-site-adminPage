# 펀딩 사이트 (관리자 페이지)
3인 조별 과제, 관리자 페이지 담당

- 펀딩사이트에서 유저가 신청하는 프로젝트를 승인 또는 거부 할 수 있는 기능을 가지고 있다.
- 가입한 계정의 정보를 볼 수 있다.
- 공지사항을 작성하여 펀딩사이트에서 유저들이 공지사항을 볼 수 있도록 한다.
